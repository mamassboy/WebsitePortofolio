# Professional Portfolio - Dimas Arya Arjuna

Website personal branding dan portofolio profesional yang modern, responsif, dan bilingual.

## Fitur Utama
- **Bilingual (ID/EN)**: Toggle bahasa yang tersimpan di localStorage.
- **Dark Mode**: Tema gelap/terang yang tersimpan di localStorage.
- **Dynamic Content**: Data proyek, sertifikat, dan keahlian dikelola terpusat di `js/data.js`.
- **Responsive Design**: Mobile-first menggunakan Bootstrap 5 grid system.
- **Project Filtering**: Filter proyek berdasarkan kategori dan fitur pencarian.
- **PDF Preview**: Tinjau sertifikat langsung di browser melalui modal.
- **Contact Form**: Integrasi siap pakai dengan Formspree atau EmailJS.

## Teknologi
- HTML5 & CSS3 (Vanilla)
- Bootstrap 5
- JavaScript (Vanilla ES6)

## Cara Menjalankan Lokal
1. Download atau Clone repository ini.
2. Buka folder proyek di teks editor pilihan Anda (misalnya VS Code).
3. Gunakan extension **Live Server** di VS Code atau cukup buka `index.html` langsung di browser.

## Cara Deploy ke Vercel (Langkah demi Langkah)
1. Buat akun di [Vercel](https://vercel.com).
2. Install Vercel CLI via npm:
   ```bash
   npm install -g vercel
   ```
3. Di root folder proyek, jalankan perintah:
   ```bash
   vercel
   ```
4. Ikuti instruksi di terminal (pilih 'Yes' untuk setup proyek baru).
5. Tunggu proses deployment selesai. Website Anda akan online dalam hitungan detik!
